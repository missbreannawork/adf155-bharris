# Accessible Soundboard 🎧

## Project Overview
This project is an accessible web-based soundboard that allows users to trigger sounds using both mouse clicks and keyboard input.


## Final Milestone

- Fixed audio file paths
- Improved UI styling
- Cleaned repository
- Added accessibility instructions
## Features
- Keyboard controls (K, G, J, D, L, H)
- Clickable sound buttons
- "Now Playing" status update
- Accessible layout and navigation
- Organized file structure

## How to Use
- Click any button to play a sound
- Or press keys: K, G, J, D, L, H
- Watch the "Now Playing" section update
