# Accessible Interface Soundboard

## Overview
This project is an accessible soundboard interface built using HTML, CSS, and JavaScript. The goal of the application is to provide a simple soundboard that can be used with both mouse interaction and keyboard controls to support accessibility.

Users can trigger sounds by clicking buttons or using keyboard shortcuts. The interface is designed to demonstrate how accessibility considerations can be incorporated into interactive web applications.

## Features
- Keyboard accessible sound controls
- Button triggered audio playback
- Simple and clean user interface
- Organized audio file management
- Responsive button layout

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Navigate to the accessible-interface folder
3. Open `index.html` in a browser or use Live Server in VS Code
