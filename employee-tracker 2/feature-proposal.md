# Feature Enhancement Proposal – Employee Tracker

## Feature Title
Improved Data Management + Validation + Sorting

## Description
This enhancement will improve the employee tracker by adding better input validation, sorting functionality, and cleaner output formatting in the console.

## Goals
- Prevent invalid user input
- Improve readability of console output
- Add sorting and filtering capabilities
- Enhance user interaction flow

## Features to Implement
- Input validation for name, age, and pay
- Sort employees by salary
- Search employee by name
- Improve console formatting with spacing
- Add confirmation messages for actions

## Value to Portfolio
This project demonstrates object-oriented programming, data handling, and user input management.
